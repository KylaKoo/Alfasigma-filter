Filters Tool

Controls on the left · Square canvas on the right (2000×2000 export)

1) Upload image

A default image is loaded automatically. Upload your own image to replace it.

2) Choose filter

Filter 1 · Controls

Original image is painted first (no black). Minimum intensity is very light.

Filter 2 · Controls

Background is CMYK-processed, then randomly stacks 1 ~ Max extra layers using DIFFERENCE blend.

Filter 3 · Shader

Move the mouse over the canvas to trigger short “running” bursts. No sliders. Export matches at 2000×2000.

Filter 4 · Collage + CMYK + Blur

Uses main image as background. Randomly crops from image #1/#2/#3, overlays pieces; each piece may optionally get a CMYK coloring and/or blur.

3) Export

Renders to an offscreen 2000px canvas using current settings.
First export can take a moment.
